## Description

Soum Microservice Template is TypeScript based repository.

## Installation

```bash
$ npm install
```

## Build

```bash
$ npm run build
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## gRPC communication

```bash
# to generate proto files
$ npm run proto
```

## Running grpc app

```bash
# development
$ npm run grpc

# watch mode
$ npm run grpc:dev

# production mode
$ npm run grpc:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
